This file contains the source code of some examples of modules that can be
associated with UGC's that you have in the folder.

ATTENTION: Visual Studio 2022 is required to correctly compile the project

MonkeyIslandWheelModule:
- MonkeyIslandWheelController: Simple example to create a rotating wheel using controls
MultiSampleModule: Includes various examples
- AnimatedPlaneController: Associate it to the example UGC so that it flies around the room.
- ZombieController: Associate it to the UGC so that your zombie walks around the room.

Once compiled they should automatically be created in the WIGUx\Modules folder

To associate the modules with UGC's you need to use the Content Manager
